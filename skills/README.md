# Managed Skills
IMPORTANT!  THESE SKILLS ARE FOR THE BINARY TO USE TO INSTALL INTO A PROJECT FOR PROJECTS THAT USE KNOTS.  THESE SKILLS ARE NOT MEANT TO BE USED TO SUPPLY TO AGENTS AS PART OF INVOCATIONS LIKE `kno claim`.

Simply put, these skills define how to use the `kno` cli only.  They DO NOT define how to execute workflow steps.  All workflow steps and prompts must be defined only in loom.
